Java

Java is a object-oriented programming language created in 1995 by James Gosling and Patrick Naughton and has been designed to be extremly portable. Actually, java is one of the most popular programming language with a reported 9 millions developers.

Get Started

And in top of that, it’s quite simple to get started ! First of all, you’ll need an IDE (Integrated Development Environment) in order to write your source code. We suggest you to use Eclipse witch can be downloaded here : https://eclipse.org/downloads/.

As soon as the download process is over, unzip the Eclipse files where you want to. Then enter the new directory and launch Eclipse.

Eclipse will first ask you to specifiy the location of your futures projects.

Once you’ve selected your save directory, you’ll have to start a New Project. To do this click on File > New > Java Project.
It’s now time to configure your project. Just select a Name for your project then click on Finish (don’t worry about others options they will be explained in an other course).

You now may have the same window as this one :

Your project is ready, now you have to creater a new class in order to start. To do this click on File > New > Java Class.
In this window, we have highlighted a few part who are important for you code.

The first green field is the folder where your class will be added.
The second green field is the name of you class. And finally the third green field regroup some options that are important. For now just check the “public static void main(String[] args)” button in order to specify that this class will be the entry point of your program.

You are now ready to start your first java program. In the next section we will present you the existing java types for variables.
