Variables and operators

In java, a variable is declared like this :
"type of variable" "name of variable" ;
There are two types of variables:

  • simple variable
  • complexe variable (objects)
Numerical variables : Java contain various type of numerical variable. The most used are int, float and double. 

float number;
number = 2.0f;

Variable storing a character :
The char type contain a character between apostrophe : 

char c;
c = 'A';

Boolean value : The boolean type, is a type who contain only two values : true or false. 

boolean question;
question = true;

And String type: The String type manage the storage of text. This is a variable of a type more complex called object . You will see that it is used a little differently from previous variables. Indeed, an object can be declared in several ways.

Like i said, String is an object and for object we talk about an instance of String.

Standard Input/Output

Using Java type and variable is very useful but you may want to interact with your program. For example printing result in your terminal or affecting value to a variable during execution.
To do this, you have to use the standard Input/Output of java.

Standard Output : 

System.out.print(“String you want to print”);
System.out.println(“String you want to print with a carriage return”);

Let’s explain those two lines.
“System” is a Java utility class who manage the standard input/output
“out” is an object of the “System” class to use the standard output
“print” method of the “out” object who print the String in parameter into the terminal
“println” Same as “print” with a carriage return at the end of the String

Standard Input : 

import java.util.Scanner; // On top of your program
Scanner sc = new Scanner(System.in);
String str = sc.nextLine();

“import” is a very important instruction which allow you to use some class who already exist in java. Basically, you import the class you want in order to use it in your class. Here we import the class “Scanner” who manage the java Standard input.
“Scanner sc = new Scanner(System.in);” Here we create a new object “sc” of the “Scanner” class using the parameter “System.in” (for the standart input).
Then we use the method “nextLine()” of this object to read the next String who will be write by the user into the terminal.

He may not be very clear so let’s take an example : a program who ask the name of the user and then say “Hi”.

import java.util.Scanner;

public class Example{
	public static void main(String[] args){
		System.out.println(“Enter your name”);
		Scanner sc = new Scanner(System.in);
		String str = sc.nextLine();
		System.out.println(“Hi ”+str);
	}
}

Of course the class “Scanner” contains many method. “nextLine()” only read the String. To read other type of variable you may use “nextVariable();” (Variable is the type beginning with a capital letter); For example, to read an integer you should use “nextInt()”, to read a boolean : “nextBoolean()” etc…

You can also found all methods of the “Scanner” class on the Oracle documentation : http://docs.oracle.com/javase/7/docs/api/java/util/Scanner.html



Condition

Condition control the order of execution in a java program, based on data values and conditional logic. There are three main categories of condition : if, if-else and switch. This condition need conditional logic operator based on boolean logic. You need to know this :

  • « == » : allow to try equality.
  • « != » : allow to try inequality.
  • < » : strictly less.
  • « <= » : less or equal.
  • « > » : strictly greater.
  • « >= » : greater or equal.
  • « && » : operator AND
  • « || » : operator OR

The if statement executes a block of code only if the specified expression is true. If the value is false, then the if block is skipped and execution continues with the rest of the program. You can either have a single statement or a block of code within an if statement. Note that the conditional expression must be a Boolean expression.

int a = 10, b = 20;
if (a > b){
	System.out.println("a > b");
}

The if/else statement is an extension of the if statement. If the statements in the if statement fails, the statements in the else block are executed. You can either have a single statement or a block of code within if-else blocks.

int a = 10, b = 20;
if (a > b) {
	System.out.println("a > b");
} else {
	System.out.println("b > a");
}

The switch case statement, also called a case statement is a multi-way branch with several choices. A switch is easier to implement than a series of if/else statements but the result it’s the same. The switch statement begins with a keyword, followed by an expression that equates to a no long integral value.

char c = ‘a’;
switch (c) {
	case ‘a’:
		System.out.println("first letter of alphabet");
		break;
	case ‘b’:
		System.out.println("second letter of alphabet");
		break;
	case ‘c’:
		System.out.println("third letter of alphabet");
		break;
	default:
		System.out.println("Cannot be determined");
}

Loop

Like the condition, the loop can control the order of execution in a java program. There are three category of loop : while, do-while and for.

The while statement is a looping construct control statement that executes a block of code while a condition is true. You can either have a single statement or a block of code within the while loop. The loop will never be executed if the testing expression evaluates to false. The loop condition must be a boolean expression.

int i = 1;
while (i<= 10) {
	System.out.println(i++);
}

The do-while loop is similar to the while loop, except that the test is performed at the end of the loop instead of at the beginning. This ensures that the loop will be executed at least once. A do-while loop begins with the keyword do, followed by the statements that make up the body of the loop. Finally, the keyword while and the test expression completes the do-while loop. When the loop condition becomes false, the loop is terminated and execution continues with the statement immediately following the loop.

int i = 1;
do {
	System.out.println(i++);
} while (i <= 10);

The for loop is a looping construct which can execute a set of instructions a specified number of times. It’s a counter controlled loop.

The syntax of the loop is as follows:

for (initialization>; loopCondition>; incrementExpression)
{
	loopBody
}

The first part of a for statement is a starting initialization, which executes once before the loop begins. The second part of a for statement is a test expression. As long as the expression is true, the loop will continue. If this expression is evaluated as false the first time, the loop will never be executed. The third part of the for statement is the body of the loop. These are the instructions that are repeated each time the program executes the loop. The final part of the for statement is an increment expression that automatically executes after each repetition of the loop body.

for (int i = 1; i <= 10; i++) {
	System.out.println(i);
}

Exercise : Conversion between Celsius and Fahrenheit

You now know how to use variables, conditions and loops and it’s time to use your knowledge in a simple exercise.
The small program we want you to make is a Celsius-Fahrenheit converter.
To do this program you’ll have a few constraints :

  • The program has to ask you which conversion you want to do (Celsius-Fahrenheit or Fahrenheit-Celsius)
  • After the conversion, the program has to ask if you want to do an other conversion or not

Of course, we’ll give you the mathematical formula to convert Celsius to Fahrenheit :
F = 9/5 * C + 32
And in the other way :
C=((F-32)*5)/9
That’s all !
